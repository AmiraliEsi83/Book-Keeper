<h2 id="about-the-project"> :pencil: About The Project</h2>

<p align="justify">
A app that allows users to save bookmarks in browser, and stores in the <a href="https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage" target="_blank">Window.localStorage</a>.
</p>
